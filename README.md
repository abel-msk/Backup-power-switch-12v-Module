#Power backup board

Board for 12v power switch.  Plain power and battery backup.  Switchng provided by LM393 comparator. Both compared   line are adjustable by potentiometer.
Equipped with:
Battery 3S charger, 
Two INA219 i2c sensors
ESP32-C3 wifi board
Charge and done charger passed to ESP
External connector for i2x line and charger status
